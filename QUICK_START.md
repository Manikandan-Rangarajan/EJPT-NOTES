## 🐛 Troubleshooting

### Theme Not Saving?
- Clear browser cache
- Check if localStorage is enabled
- Try incognito/private mode

### Filters Not Working?
- Make sure you're using current browser version
- Try refreshing the page
- Check browser console for errors (F12)

### Links Not Opening?
- Check your internet connection
- Links open in new tab/window
- Allow pop-ups if blocked

### Search Not Finding Tools?
- Make sure to search exact tool names
- Try partial matches (e.g., "nmap" or "nma")
- Combine with difficulty filter

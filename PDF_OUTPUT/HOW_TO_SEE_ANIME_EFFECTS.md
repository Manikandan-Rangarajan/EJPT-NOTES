## Common Questions

### Q: I don't see any changes!
**A:** Make sure you:
1. Have `anime-vibes.css` file in the same directory as the HTML files
2. Refresh the browser (Ctrl+F5 or Cmd+Shift+R)
3. Try a different browser
4. Check browser console (F12) for errors

### Q: Code still looks boring
**A:** Try:
1. Scroll to find actual code examples (with `<pre>` tags)
2. Make sure anime-vibes.css is loaded (check in browser's Developer Tools → Network tab)
3. Try a different HTML file with more code examples

### Q: Theme toggle doesn't work
**A:** Make sure:
1. JavaScript is enabled in your browser
2. localStorage is available (not blocked)
3. Try clearing browser cache
4. Try in a different browser

### Q: Colors look different on my device
**A:** This is normal! Color rendering varies by:
- Monitor/screen calibration
- Display settings
- Browser color management
- Light/dark mode preferences

### Q: Mobile version looks wrong
**A:** Try:
1. Check browser zoom level (should be 100%)
2. Rotate device to landscape/portrait
3. Try a different mobile browser
4. Clear browser cache

## Browser DevTools Tips

### To verify CSS is loaded:
1. Press F12 to open Developer Tools
2. Go to "Elements" or "Inspector" tab
3. Look for `<link rel="stylesheet" href="anime-vibes.css">`
4. It should show as green (loaded successfully)

### To see all CSS classes available:
1. Open `anime-vibes.css` in a text editor
2. Search for class names like `.btn-primary`, `.card`, `.alert`, etc.
3. You can use these classes in your HTML

### To test theme toggle:
1. Open console (F12 → Console tab)
2. Type: `document.body.classList.toggle('light-mode')`
3. Theme should change instantly

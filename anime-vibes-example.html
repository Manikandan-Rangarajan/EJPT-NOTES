💻 Code Examples

Using Anime Vibes CSS

Link the CSS file in your HTML head:

<link rel="stylesheet" href="anime-vibes.css">

Then use classes in your HTML:

<div class="card anime-glow">
  <div class="card-header">Title</div>
  <div class="card-body">Content</div>
</div>

Use buttons with variants:

<button class="btn btn-primary">Action</button>
<button class="btn btn-secondary">Option</button>

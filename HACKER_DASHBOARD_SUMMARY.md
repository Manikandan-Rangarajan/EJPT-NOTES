## 🎨 Color Customization Guide

To change colors, edit the `:root` CSS variables:

```css
:root {
    --primary-color: #00ff00;      /* Change green */
    --secondary-color: #00cc00;    /* Change hover green */
    --accent-color: #ff0099;       /* Change pink */
    --dark-bg: #0a0e27;           /* Change background */
    --darker-bg: #050812;         /* Change darker bg */
    --text-color: #e0e0e0;        /* Change text */
    --border-color: #00ff00;      /* Change borders */
}
```

For example, purple theme:
```css
--primary-color: #aa00ff;
--accent-color: #00ffff;
--dark-bg: #1a0033;
```

## 📖 User Guide

### Using Dark/Light Mode
1. Click 🌙 Dark Mode or 🌞 Light Mode button (top-right)
2. Theme changes instantly
3. Your preference is saved automatically

### Using Difficulty Filters
1. Click difficulty buttons: All Levels, Beginner, Intermediate, Advanced
2. Cards automatically filter
3. Combine with search for precision

### Using Learning Paths
1. Click "📚 Learning Paths" button
2. Select a learning path from the modal
3. Click any tool tag to jump to search
4. Press Escape to close

### Using Resource Links
1. Look for 🔗 buttons on tool cards
2. Click to open official documentation
3. Links open in new tab
